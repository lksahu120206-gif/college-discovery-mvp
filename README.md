# EduExplore Platform (Multi-Track Project Showcase)

A multi-page project built with Next.js 15, TypeScript, and Tailwind CSS. This repository contains three distinct project modules built to handle dynamic UI rendering, data filtering, and metric comparisons.

**🔗 Live Production Link:** [https://college-discovery-mvp-bay.vercel.app/](https://college-discovery-mvp-bay.vercel.app/)

---

## 🧭 Live Project Modules

The application is split into three separate routes. You can navigate between them live:

### 1. Track B: College Discovery Platform (Main Hub)
* **Live Link:** [https://college-discovery-mvp-bay.vercel.app/](https://college-discovery-mvp-bay.vercel.app/)
* **Features:** An instant search bar to filter through university cards by name or location, plus an interactive side-by-side comparison matrix for comparing multiple campuses.

### 2. Track A: Dynamic JSON UI Generator
* **Live Link:** [https://college-discovery-mvp-bay.vercel.app/generator](https://college-discovery-mvp-bay.vercel.app/generator)
* **Features:** A live sandbox engine that reads raw JSON configurations and dynamically renders input fields, dropdowns, and buttons on the fly. It includes error boundaries so that unexpected or broken JSON tags display a warning badge instead of crashing the page layout.

### 3. Track C: Tech Compensation Matrix (Levels.fyi Clone)
* **Live Link:** [https://college-discovery-mvp-bay.vercel.app/compensation](https://college-discovery-mvp-bay.vercel.app/compensation)
* **Features:** A data ledger that maps engineering levels across companies like Google, Meta, and Amazon. It breaks down base salary, equity, and bonuses, with interactive sorting logic to calculate total compensation components instantly.
