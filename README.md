# College Discovery MVP (EduExplore)

This is a simple college discovery platform project that I built and deployed. It lets users search through campuses, filter by tuition fees, location, and compare ratings using a clean card UI.

Live Demo: https://college-discovery-mvp-bay.vercel.app/

## Tech Stack
* Next.js / React
* TypeScript
* Tailwind CSS
* Deployed on Vercel

## Features
* Search bar to filter through different campuses instantly
* Card layout displaying tuition fees, location, and ratings
* Fully responsive layout for mobile and desktop screens
* Auto-deployment (CI/CD) hooked up via GitHub and Vercel
